0. [Install Homebrew](https://brew.sh)
1. Install Python using Homebrew
  ```bash
  brew install python@3.11
  ```

2. Install package manager uv
```bash
pip3.11 install uv
```

3. Create venv using uv
```bash
uv venv --python=python3.11
```

4. Activate env
```bash
source .venv/bin/activate
```
